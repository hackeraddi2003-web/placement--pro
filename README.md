# 🎯 The Ultimate Personal Placement Preparation Operating System

**PlacementOS Pro** is a modern productivity and placement preparation
platform built for students preparing for internships and placements.

It combines **DSA Tracking, English Practice, Smart Journaling, Project
Management, Interview Preparation, Job Tracking, AI Mentor, Analytics,
Gamification, Cloud Sync and Offline Support** into one unified
dashboard.

# 🏆 Gamification

## XP System

  Activity           Reward
  ------------------ --------
  Daily Task         +10 XP
  Goal Completed     +20 XP
  Journal            +20 XP
  DSA Problem        +15 XP
  English Practice   +15 XP
  New Project        +30 XP

## ❤️ Hearts

-   Start with **3 Hearts**
-   Missing study breaks streak
-   Losing all hearts causes **-50 XP**
-   Complete Daily Quest to recover hearts

## 🎯 Daily Quest

Complete **2 Daily Tasks**

Reward:

-   +25 XP
-   +1 Heart ❤️
